Duplo Sim (Double Yes)

A system to allow clients to remotely contract an insurance policy and then confirm or discard it later.

As required by the Brazilian law, clients should be able, inside a certain period of time, to confirm or discard an insurance policy hired remotely. The Duplo Sim system is a system I helped build and design to allow them to do just that. We used Cobol, JCL, IBM utilities, SQL (DB2 database), Java (to access the Cobol backend using a technology called “containers”), Angular (for the web application frontend) and native Android APIs for the mobile application.

  • Date: 2016
  • Client: Banco do Brasil
  • Category: Insurance

Maintenance of the Insurance System

General maintenance of the Bank’s insurance system.

Many problems arise from any automated system’s operations, and the Bank’s insurance system is no different. Using many tools at my disposal, I investigated and solved problems involving wrong processing of sequential files, bugs encountered by users, missing data from the databases, mistaken reversal of paid premiums, among many others.

  • Client: Banco do Brasil

Ourovida Garantia

An insurance product. The insured has his/her debts paid off in case of death or permanent disablement.

A life insurance whereby the insured has his/her debts paid off in case of death or permanent disablement. It was built using Cobol, JCL, SQL, NATURAL, Java (to access the Cobol backend using a technology called “containers”), Angular (for the web application frontend) and native Android APIs for the mobile application.

  • Date: 2015
  • Client: Banco do Brasil
  • Category: Insurance

Product Versioning

A system to allow non-technical product managers to easily update product characteristics, abstracting the underlying databases.

Before this project, product managers had to contact the Bank’s technology division to update product characteristics. Now, they can do it by themselves and immediately test the changes, greatly helping their jobs. This project was built with Cobol, SQL and a proprietary language to build user interfaces called NATURAL.

  • Date: 2016
  • Client: Banco do Brasil
  • Category: Insurance
